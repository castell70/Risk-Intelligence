Información breve, objetivo y resumen de indicadores
Introducción

Esta aplicación visualiza el índice de riesgo de deserción estudiantil y permite explorar datos por facultad, carrera y estudiante para apoyar decisiones académicas.

Objetivo

Proveer una vista rápida e interactiva del semáforo de riesgo, distribuir y filtrar registros, y detectar inconsistencias en los datos fuente.

Paso a paso para usar la herramienta
  1. Preparar archivo: Asegúrate de que el archivo .xlsx tenga las columnas requeridas (nombre, facultad, carrera y los indicadores descritos en el documento).
  2. Cargar datos: Ve a Herramientas → Seleccionar archivo → Cargar datos. Espera la confirmación de procesamiento.
  3. Explorar Dashboard: Revisa la distribución de riesgo y el radar de indicadores en la sección Dashboard.
  4. Filtrar registros: Usa los filtros por facultad, carrera, búsqueda por nombre y nivel de riesgo en la sección Estudiantes para acotar la lista.
  5. Ver detalle de un estudiante: Haz clic en una fila de estudiante para abrir el detalle y revisar indicadores individuales (las métricas 'Riesgo' se muestran como porcentaje cuando aplica).
  6. Verificación de datos: Revisa la sección Verificación para ver errores y advertencias detectadas y corregir el archivo fuente si es necesario.
  7. Iterar: Corrige y vuelve a cargar el archivo según las advertencias para obtener resultados más confiables.
Resumen del documento de indicadores

El documento explica la metodología paso a paso para construir el Índice de Riesgo de Deserción (IRD): primero se recogen indicadores clave por estudiante (promedio académico, asistencia y número de asignaturas aprobadas) y se normalizan para hacerlos comparables. El promedio académico se transforma mediante z-score (desviación respecto a la media de la cohorte) y se invierte para que valores bajos de rendimiento incrementen el riesgo; asistencia y asignaturas aprobadas se normalizan por min-max para obtener valores entre 0 y 1 y luego se invierten cuando corresponda (por ejemplo, baja asistencia = mayor riesgo). Cada componente recibe un peso (ej. rendimiento 40%, asistencia 30%, aprobadas 30%) y se combinan en una única puntuación IRD que se utiliza para clasificar en tres niveles: Bajo, Medio y Alto (umbrales definidos en el documento). Además se describe una columna textual complementaria (Sem_Z) que, si está presente y contiene etiquetas como "Alto/Medio/Bajo", se prioriza como etiqueta final; en caso contrario se usa el IRD para asignar el nivel. El PDF incluye las convenciones de nombres de columnas esperadas, códigos de variables, ejemplos de limpieza de datos (tratamiento de valores faltantes, formatos numéricos y valores atípicos) y recomendaciones para preparar el archivo antes de cargarlo en la aplicación.

Sobre el informe "Seguimiento por períodos"

Este informe genera un listado filtrado por Facultad, Carrera y Nivel de Riesgo que sirve para el seguimiento de estudiantes en tres períodos. Use "Mostrar listado" para visualizar en pantalla los registros que cumplen los filtros seleccionados.

Botones disponibles:

  • Mostrar listado: muestra en pantalla los estudiantes filtrados.
  • Exportar estadísticas (Word): descarga un documento con el resumen y gráficos del Dashboard (útil para informes rápidos).
  • Exportar listado (Excel): descarga el listado filtrado con las columnas de firma y seguimiento (Firma1/Seguimiento1/Observaciones1 ... Firma3/Seguimiento3/Observaciones3) para uso en hojas de trabajo.
