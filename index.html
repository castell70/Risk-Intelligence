Introducción breve

La herramienta procesa respuestas tipo Likert y numéricas para generar un índice compuesto (0–10) que ayuda a priorizar intervenciones por estudiante y por facultad.

Cómo se realizan los cálculos (detalle y ejemplo)
  1. Clasificación de preguntas por dominio usando KEYWORDS; columnas con NEGATIVES mantienen dirección de riesgo.
  2. Mapeo Likert -> 1-5 y normalización a escala 0–5 (neutral ≈2.5). Valores numéricos válidos también se usan.
  3. Si la pregunta es positiva se invierte la dirección de riesgo en la escala 0–5; preguntas negativas mantienen dirección.
  4. Promedio por dominio (0–5). Si no hay ítems se usa punto medio 2.5.
  5. Banderas rojas (redFlags) detectan señales críticas y suman +0.3 cada una al puntaje en 0–5.
  6. Puntaje final = suma ponderada de dominios + ajuste banderas (clamp 0–5) → multiplicar por 2 → escala 0–10.
Ejemplo: dominios = {A:4.0(15%), P:3.5(35%), S:2.0(20%), E:3.0(20%), H:2.5(10%)} → base≈3.14 + 0.3(redFlag)=3.44 → 6.9/10 → Nivel Medio.
Interpretación y Umbrales

Escala 0–10:

  • Bajo: < 4.0
  • Medio: 4.0 – 6.9
  • Alto: ≥ 7.0 (o ≥2 redFlags)

Nota: umbrales ajustados para priorizar casos críticos; las banderas rojas elevan el puntaje para asegurar visibilidad.

Ejemplo detallado de transformación Likert → riesgo

Mapeo: "Totalmente de acuerdo" = 5 → normalizado = 5.0 en 0–5; si la pregunta es POSITIVA se invierte a (5 - 5.0) = 0 (bajo riesgo).

Si la pregunta es NEGATIVA, "Totalmente de acuerdo" = 5 → normalizado = 5.0 → riesgo = 5.0 (alto).

Cómo se calcula la Salud Institucional

La "Salud Institucional" mostrada en el dashboard es un resumen agregado del nivel de riesgo promedio de toda la población en escala 0–100% derivado del promedio de puntajes globales (0–10).

  1. Se calcula el puntaje por estudiante en escala 0–10 (ver proceso de dominios, pesos y banderas).
  2. Se obtiene el promedio aritmético de esos puntajes (ej: suma de todos los score10 / N).
  3. Ese promedio (0–10) se multiplica por 10 para convertirlo a porcentaje (0–100%).
  4. El KPI "Salud Institucional" muestra ese porcentaje redondeado y sirve para comparar periodos o subgrupos (facultad/carrera).
Ejemplo: Si la muestra tiene 5 estudiantes con scores: 8.0, 6.5, 5.0, 7.5, 4.0 → promedio = (8+6.5+5+7.5+4)/5 = 6.2 → Salud Institucional = 6.2*10 = 62%. En el dashboard aparecerá "62%".

Nota: al aplicar filtros (facultad/carrera) el KPI se recalcula con la subpoblación filtrada; si hay pocos registros, interpretar con precaución.

Opciones Rápidas
  • Importar .xlsx: primera hoja, mapeo automático.
  • Filtrar por Facultad y Riesgo en la tabla.
  • Exportar Resultados: XLSX con resumen por estudiante.
Consejos de Calidad
  • Revise encabezados: incluir palabras clave de KEYWORDS mejora el mapeo.
  • Evite celdas vacías: las preguntas ausentes se tratan como neutras (2.5).
  • Use respuestas consistentes tipo Likert para mejores promedios.
